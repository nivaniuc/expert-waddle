## Introduction
This is a MySQL database that uses Node.js to run the backend of a E-Commerce site.

## Install

npm init

npm install inquirer

npm install mysql

## Usage

mysql -u root -p
enter password  
source db/schema.sql  
quit out  
npm run seed  
npm start
